# Lovelace VMI Ventilairsec

Copiez le contenu du fichier [lovelace/vmi_ventilairsec_dashboard.yaml](lovelace/vmi_ventilairsec_dashboard.yaml) dans l’éditeur de tableaux de bord de Home Assistant.

## Notes
- Les entités utilisées dans ce dashboard correspondent aux entités de base créées par l’intégration custom.
- Si vous avez renommé les entités, adaptez les noms dans le fichier YAML.
